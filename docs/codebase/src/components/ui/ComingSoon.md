## Purpose

Placeholder component for features under development. Shows a title, description, optional expected date, and optional back button.

| `description` | `string` | ❌ | Default Hebrew | Feature description |
| `expectedDate` | `string` | ❌ | — | Expected availability date |
| `showBackButton` | `boolean` | ❌ | `true` | Show back button |
