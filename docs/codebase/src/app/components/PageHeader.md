# PageHeader.tsx

**File:** `src/app/components/PageHeader.tsx`
**Status:** Active

## Purpose

Renders the page title and optional subtitle below the top bar, inside the main content column. Owned by `AppShell` — pages don't render this directly.

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `title` | `string` | ✅ | Page headline |
| `subtitle` | `string` | — | Optional one-line description |
